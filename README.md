#  Installation
1. go to super user
```bash
sudo su
```
2. clone directory
```bash
git clone https://github.com/Ntf-Sadnan/ntf.git 
```
3. go to the directory
```bash
cd ntf 
```
4. give permissions
```bash
chmod +x * 
```
5. run the `install` file 
```bash 
./install 
```

# RunMe
NOTE: run as super user always!
